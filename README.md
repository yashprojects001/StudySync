## 🌈 Overview

**StudySync** is a modern student productivity platform built to help students stay focused, organized, and consistent.

It combines:
- ⏳ **Pomodoro Timer**
- 📅 **Timetable Planner**
- 🔔 **Assignment Reminders**
- 📝 **Collaborative Notes**
- 📈 **Progress Heatmap**

And many more smart study tools.

---

## 🚀 Features

### 🎯 Core Features
- **Pomodoro Timer** with focus / break cycles
- **Timetable Planner** for daily class scheduling
- **Assignment Reminders** for due dates and tasks
- **Collaborative Notes** for shared study material
- **Progress Heatmap** to track consistency

### ✨ Extra Features
- ✅ To-do checklist
- 🧠 Flashcards section
- 🎯 Daily study goal tracker
- 📘 Exam countdown
- 📊 Subject progress tracker
- 🎵 Focus mode selector
- 🌟 Inspirational quote generator
- 🚫 Distraction blocker
- 💾 Local browser storage support
